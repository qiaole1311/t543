# T543 - Interactive Educational Materials

This repository contains interactive quizzes and educational materials for the T543 module.

## Quizzes

- [Nucleotide Structure Quiz](quizzes/nucleotide-structure.html) - Learning Goals Checkpoint: structure of nucleotides

## Adding to Google Sites

To embed any quiz in Google Sites, use:

```html
<iframe src="https://qiaole1311.github.io/t543/quizzes/nucleotide-structure.html" width="100%" height="600" frameborder="0"></iframe>
```

Adjust the `src` URL and dimensions as needed for different quizzes.
